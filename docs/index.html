Physics for Programmers

These pages explain physics. They are intended to be used by programmers and contain many interactive visual aids.

The abstract concepts of physics have been somewhat simplified, so as to not confuse the average programmer.

Those lacking programming skills or needing a refresher need not worry, since we also go over basic programming concepts.
